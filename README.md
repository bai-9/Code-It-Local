# Code-It-Local
A local version of Code-It shiny app

## Overview
This is local verison of Code-It shiny app. The web app is available at
http://codeitapp.org.

## Project Structure
This project follows a modular Shiny structure:
* `ui.R` / `server.R` / `global.R`: Core Shiny logic.
* `/R`: Helper functions and plot scripts.
* `/tmp`: Local data storage.


## How to Run Locally
1. Open the `Code-It-Local.Rproj` file in RStudio.
2. Ensure you have the required libraries installed (see global.R).
3. Open `global.R` or `server.R`.
4. Click the **Run App** button in the top right of the editor pane.
